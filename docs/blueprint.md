# **App Name**: FinTrack

## Core Features:

- Expense Input: Add expenses with categories (e.g., food, transport).
- Budget Setting: Set monthly budgets for different categories.
- Spending Visualization: Interactive charts to visualize spending patterns.
- Dashboard Overview: Display key financial insights on a clean dashboard.
- Smart Insights: AI-powered recommendations for budget adjustments based on spending habits, using its analysis as a tool.

## Style Guidelines:

- Primary color: Dark blue (#2C3E50) to convey trust and stability.
- Background color: Light gray (#F0F4F8) for a modern and clean look.
- Accent color: Teal (#008080) for highlighting key elements and actions.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and warm feel, suitable for both headlines and body text.
- Use minimalistic icons to represent expense categories and other functions.
- Clean, card-based layout for easy navigation and information access.
- Subtle transitions and animations to enhance user experience when navigating the app or viewing charts.
